# rust-hyper-toggle

Testing rust and hyper (HTTP lib) by building a minimal api proxy over Toggle.io API.

## Run with watch

> Requires `cargo-watch`, install via `cargo install cargo-watch`

```sh
cargo watch -c -w src -x run
```
